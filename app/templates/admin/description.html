{{ _('About Excel Chat Assistant') }}

Excel Chat Assistant - это платформа для анализа и работы с данными Excel при помощи искусственного интеллекта. Система позволяет загружать Excel-файлы, задавать вопросы о данных и получать инсайты в формате диалога.

{{ _('Key Features') }}
  • Загрузка и управление Excel-файлами
  • Чат-интерфейс для взаимодействия с данными
  • Три режима доступа: чтение, чтение/запись, полный доступ
  • Многоязычный интерфейс (английский, немецкий, русский)
  • Управление пользователями и подписками
  • Аналитика использования системы
